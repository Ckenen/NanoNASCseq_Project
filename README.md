# Workstation of Nanopore-Based Newly Synthesized RNA Sequencing

This repository include all the **Source Code** for the entire project. In addition to the analysis code for **NanoNASC-seq**, it also includes the analysis code for **RNA-seq**, **NACS-seq**, **scCOLOR-seq**, **SCAN-seq2**, **PRO-seq**, **TT-seq**, **NanoCOP**, and even the **Plotting Code** for figures in the article. They are organized into different **Sub-Directories**.

Please find the detailed description for the specific method in the corresponding sub-directory.

## Contents

| Sub-Directory | Description |
| :-- | :-- |
| 0_Analysis | Source code of advanced analysis and plotting **Figures**. |
| 1_NanoNASCseq | The analysis of **NanoNASC-seq**. |
| 2_NASCseq | The analysis of **NASC-seq**. (Include datasets form **TangLab** and **GSE128273**) |
| 3_RNAseq_ActD | The analysis of **RNA-seq** under **ActD treatment**. |
| A01_PROseq_K562 | The analysis of **PRO-seq** datasets. |
| A02_TTseq_K562 | The analysis of **TT-seq** datasets. |
| A03_SCANseq2 | The analysis of **SCAN-seq2** datasets from **GSE203561**. |
| A06_scCOLORseq | The analysis of **scCOLOR-seq** datasets from **GSE162053**. |
| A07_GSE141851_scNTseq | The analysis of **scNT-seq** datasets from **GSE141851**.. |
| A08_GSE128365_scEUseq | The analysis of **scEU-seq** datasets from **GSE128365**. |
| A09_GSE123191_NanoCOP | The analysis of **Nano-COP** datasets from **GSE123191**. |
| envs | Conda environment definition files. |
| public | Supplementary files required by analysis and download from public database. |
| share | Share files such as genome annotation and scripts. |
